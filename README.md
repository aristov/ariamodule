# ariamodule

WAI-ARIA framework HTML DOM implementation.

_work in progress_

## Installation

```
npm install ariamodule
```

## Development

```
git clone git@github.com:aristov/ariamodule.git
cd ariamodule
npm install
npm run watch
```
